Steam Goodies:
======================

This folder contains textures for your Steam Library.


---------------------

The Steam library supports adding non-steam games. As well as giving users the ability add custom artwork for the game covers in their libraries.

Hence, I'm providing a custom library cover for Minetest; as well as the custom drawn logo icon, & website text as supplemental images.
The title is a scaled version of the 'menu_header.png'; which is already included in my texture pack.

Background pattern re-creation via GIMP: 
	1. Create a new, full-white image.
	2. Use lots of the Noise Filter, "RGB Noise"; on the white background.
	3. Make a copy of the current layer to a new layer. And, use Gaussian Blur on the new layer.
	4. Then set the Transparency of the new layer to about 80%.
	5. Adjust as needed, for the desired softness of the noise. Then merge together when ready.
	6. Use Colorize, on the final image to darken, and color the layer to your desired level.
