[SuperTuxKart Add-on Textures]:
==========================

This is a few textures, plus a script that converts, and scales the rest of my textures for use in SuperTuxKart.
Specifically, for use with Worldblender's "Minetest Sam" kart add-on.




[Instructions]:
----------------
There are a few custom textures included, and a script that uses relative pathing to copy, & imagemagick to scale my 16px textures for use with STK.


1. Run the script from this directory, and when finished you'll have a folder of 29 textures ready for use.


2. Follow these instructions on finding your add-on install location:
https://supertuxkart.net/Installing_Add-Ons


3. Then copy my 'minetest-sam' folder on top of the add-on's folder; this will merge & overwrite the original textures with my own REFI-styled textures.
